# Mon Portfolio Numérique

## Description
Bienvenue sur mon portfolio numérique ! Ce site présente mon CV, mes compétences et mes projets personnels.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript

## Comment visualiser le site
1. Double-cliquez sur `index.html` pour ouvrir dans votre navigateur
   (ou servez le dossier avec `python3 -m http.server` pour un rendu fidèle à la prod)

## Structure du projet
- `index.html` : Page d'accueil
- `alternance.html` : Page dédiée à la recherche d'alternance BUT3
- `rapport-stage.html`, `projet-ecodrop.html` : Pages de détail (stage, projet)
- `style.css` : Styles communs à toutes les pages (variables CSS, sidebar, topbar, footer)
- `doc/` : Documents supplémentaires (CV, rapports)
- `images/` : Images du portfolio
