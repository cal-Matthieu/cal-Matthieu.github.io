# Mon Portfolio Numérique

## Description
Bienvenue sur mon portfolio numérique ! Ce site présente mon CV, mes compétences et mes projets personnels.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript

## Comment visualiser le site
1. Ouvrez le dossier `html/`
2. Double-cliquez sur `index.html` pour ouvrir dans votre navigateur

## Structure du projet
- `html/index.html` : Page d'accueil
- `html/style.css` : Feuille de style
- `html/script.js` : Scripts interactifs
- `html/doc/` : Documents supplémentaires
- `html/images/` : Images du portfolio
